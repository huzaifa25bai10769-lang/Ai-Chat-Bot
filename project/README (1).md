# **🤖 Simple Python Chatbot**

A beginner-friendly rule-based chatbot built using Python. This chatbot responds to basic user inputs like greetings and simple questions.

---

## **📌 Features**

* Responds to basic greetings (Hi, Hello)  
* Answers simple questions (How are you, Your name)  
* Continuous conversation loop  
* Exit command (`bye`) to stop the chatbot

---

## **🛠️ Technologies Used**

* Python (No external libraries required)

---

## **📂 Project Structure**

chatbot.py   \# Main chatbot program  
README.md    \# Project documentation

---

## **▶️ How to Run**

1. Make sure Python is installed on your system.  
2. Download or clone this repository.  
3. Open terminal or command prompt.  
4. Run the chatbot:

python chatbot.py

---

## **💬 Example Interaction**

Chatbot: Hello\! Type 'bye' to exit.

You: hi    
Chatbot: Hi there\!

You: how are you    
Chatbot: I'm just a bot, but I'm doing great\!

You: your name    
Chatbot: I'm a simple chatbot.

You: bye    
Chatbot: Goodbye\!

---

## **⚙️ How It Works**

* The chatbot uses simple `if-elif` conditions.  
* It compares user input with predefined responses.  
* If input doesn’t match, it returns a default message.
